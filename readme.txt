Usage instructions:
1. Launch app.
2. Add a value for systolic pressure.
3. Add a value for diastolic pressure.
4. Choose a user to link the entry to.
5. Click "Add Entry".
6. Click on an individual reading to view its details.
7. Click on "Check Monthly Report" to view monthly reports.
